🧮 Basic Calculator

A simple and responsive **Calculator Web Application** built using **HTML, CSS, and JavaScript**.

This calculator performs basic arithmetic operations and includes additional features such as keyboard support and modern UI styling.

---

## 📌 Features

✅ Addition (+)  
✅ Subtraction (−)  
✅ Multiplication (×)  
✅ Division (÷)  
✅ Clear screen functionality  
✅ Delete last input  
✅ Real-time display updates  
✅ Keyboard support  
✅ Responsive and modern design

## 🧠 How It Works

1. User clicks buttons or uses keyboard input.
2. Values appear on the display screen.
3. Selected operations are added to the expression.
4. Press **=** or **Enter** to calculate.
5. Use **C** to clear or **⌫** to remove the last input.
